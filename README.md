# Mailgun with HapiJS+AMQP example

## Getting started

Clone the repository and install all dependencies through [NPM][1]

```bash
git clone https://github.com/vasim-akram/mailgun-server.git
cd mailgun-server
npm install
```

Run HapiJS server on http://localhost:8000

```bash
setup rabbitmq
```

```bash
npm start
```
